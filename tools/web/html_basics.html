HTML Alapok

Az általános felépítés:


<TAG beállítás="érték" egyszeri_érték> [...] </TAG>
Fontos megjegyezni hogy nem minden tagnek van lezárása (</TAG>)

Legfontosabb tagek:

Név megfelelő próbáld ki!
br sortörés (nincs lezárása) br
span, container, nav körbeveszi az elemet, semmit nem csinál vele br
div br és span kombinálva br
h1..h6 főcímek, H1 a legnagyobb br
input bemeneti mező. (lásd késöbb) (nincs lezárása) br
img kép (nincs lezárása) [src="" a linkeléshez] br
p bekezdés, olyan mint egy div, de előtte és utána is helyet foglal magának br
a linkel egy oldalra [href="" a linkeléshez] br
hr vízszintes vonal br
button gomb br
b / i / u / s vastag/dőlt/aláhúzott/áthúzott szöveg br


Beállítások

Név funckió használható
id="valami" azonosító mindenhol
class="valami mas" csoport azonosító, több is lehet, spacel elválasztva mindenhol
type="meghatározott" csak bemeneti mezőknél. lehet: password/number/color/date/checkbox/radio és sok más input
disabled kikapcsolt input + button
href="" hova linkeljen a, de máshol is használják
