Hello there

Welcome to the REWIKI project, which is created becase, well: Wikipedia is ugly as hell.
The main problem is that wikipedia wasn't updated in years, and not just the lack of nice design, but also the lack of useful features is disturbing.
This site aims to change that. It it based on many concepts from the internet, some parts mixed together, some parts perfected.

Please note that this app has many drawbacks.
  • It is much slower
  • It is under construction
  • lot of local wikipedias (hu, en , ge) have many different styles making it almost impossibble to style everything correctly.
