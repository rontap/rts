Objektumok

  1. HTML tagek
    a HTML tageket beírva, az összes olyan tagre alkalmazzuk ami a html-ben van.
    Pl: div { color : red; } az Összes Div színét állítja be

  2. classok
    A HTML tageknek az egyik beállítása hogy class-ot adunk nekik. (akár többet is). ezekkel csoportokba sorolhatjuk. A classok előtt egy . (pont) jelzi hogy egy classra gondolunk
    Pl: .fontos { color : green; } az Összes fontos class-ú tag színét állítja be zöldre

  3. id-k (azonosítók)
    Ugyanúgy, az id egy beállítás, azzal a külömbséggel hogy CSAK EGY lehet belőle egy tagnek, és a dokumentumnak nem lehet más, ugyanolyan nevű azonosítója. (egyértelmű hozzárendelés). Az azonosító előtt egy # (hashtag) jelzi hogy egy id-ra gondolunk
    Pl: #gomb { color : blue; } az csak egy, ezzel az azonosítóval rendelkező tag-nek színét állítja be.

Ezek ilyen sorrendben felülirják egymást. Tehát egy gomb azonosítójú divnek kék lesz a szine, mert az id felülírja az összes többi beállítást
